Tags HTML

Table

Cria uma matriz. As linhas são marcadas pela tag tr e as colunas pela tag td. Tem as propriedades
width, border, cellspacing e cellpadding.

Exemplo:

a a
a a

Del

"Risca" a palavra entre a tag.

ABCDE

Ins

Sublinha a palavra ou frase entre a tag.

ABCDE

Sub

Coloca as letras ou números mais abaixo das palavras normais.

ABCDE

Sup

Coloca as letras ou números mais acima das palavras normais.

ABCDE

Blockquote

É utilizado para citações.

Quando Basilich foi tomada pelo batalhão de Noxus, os irmãos chamaram a atenção do capitão Cyrus depois que Draven tomou a decisão nada sensata de tentar matá-lo. Impressionado com o espírito guerreiro dos dois, Cyrus permitiu que eles entrassem para o exército noxiano.

Q

É utilizado para frases curtas, diferente do blockquote, já que em blockquote ocorrem quebras de linha.

Um histórico de pesquisa vazio diz mais do que um histórico cheio.

Abbr

Usado em abreviações, quando se passa o mouse por cima é descrito o significado.

LOL

Address

Utilizado para designar endereços.

ABCDE FGHIJ KLMNO

BDO

Utilizado para deixar a frase ao contrário.

Eu amo carros esportivos.

Eu amo carros esportivos.

Code

Utilizado para designar uma linha de programação destacada na página.

Linha de programação.

Pre

O texto digitado dentro desta tag é mostrado da forma que foi escrito.
Por exemplo, se for colocado dois espaços dentro da sentença eles irão aparecer.

A B     C      D        E

KBD

É utilizado para mostrar comandos de teclas.

Ctrl + C
